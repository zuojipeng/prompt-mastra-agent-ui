# 镜词 · 核心竞争力战略

## 核心理念

> **代码可以被复制，但数据飞轮和知识体系不能。**

## 飞轮模型

```
用户生成提示词 → 去平台出片 → 回来反馈(👍/👎) 
                                        ↓
                       反馈数据存入知识库 → 分析模式/问题
                                        ↓
              Agent 学习改进 → 下轮生成质量提升
                                        ↓
                        用户更满意 → 更多使用
                                        ↓
                            (回到顶部)
```

## 两大引擎

### 引擎一：用户反馈闭环
- 每条生成结果可 👍/👎 + 文字反馈
- 反馈数据关联：输入创意、输出提示词、用户评价
- 定期分析：哪些风格/镜头/导演模式用户最喜欢
- 反馈驱动的 prompt 优化

### 引擎二：专业知识持续注入
- 定时学习：电影镜头、导演风格、摄影技巧、分镜知识
- 结构化存入知识库（已有 `~/.hermes/ai-video-knowledge/`）
- Agent system prompt 定期更新，吸收最新知识
- 知识来源：B站教程、电影分析、摄影书籍、行业案例
