# 镜词 · 产品路线图

> 产品判断：镜词应聚焦“AI 短片导演工作台”，不要继续扩成泛 prompt 工具。
> 核心目标是帮助个人创作者更稳定地把一个创意做成一条像作品的 AI 短片。

## 1. 定位

一句话：

```text
把一句创意，重构成一套更容易出片的 AI 短片执行包。
```

核心用户：
- 想做 10-90 秒 AI 短片的个人创作者
- 想做有风格、有类型感、有连续性的自媒体视频的人
- 已经使用小云雀、Seedance、Kling、Runway、Pika、Sora 等平台，但试错成本高的人

核心价值：
- 判断创意是否适合 AI 视频生成
- 把普通创意重构成更容易出片的表达
- 给出每一镜的生成策略和风险补救
- 把 prompt 变成成片工作流

## 2. 产品北极星

北极星指标：

```text
用户从一个创意拿到可执行导演包，并完成至少一次平台复制或导出的比例。
```

辅助指标：
- 创意体检完成率
- 重构版本选择率
- 导演执行包生成成功率
- 平台复制/导出率
- 点赞率 / 踩率
- 同一创意二次优化率
- 生成失败率

## 3. 路线阶段

### Phase 0: 稳定 V2 闭环

目标：
- 当前 V2 导演执行包流程稳定可发布

范围：
- 创意体检
- 三个重构版本
- 导演执行包结果页
- 平台建议
- 风险补救
- 历史和反馈不阻塞主流程

成功标准：
- 前后端契约一致
- 核心路径 E2E 通过
- 失败态可恢复
- 线上健康检查稳定

### Phase 1: 出片稳定性

目标：
- 减少 AI 视频常见翻车点

能力：
- 镜头风险标签
- 文生 / 图生 / 参考图推荐
- 主角一致性建议
- 口型、多人物、复杂动作风险规避
- 每镜失败补救方案

成功标准：
- 用户能明确知道每个镜头怎么生成更稳
- 高风险镜头有替代方案
- 平台建议不再只是泛泛说明

### Phase 2: 反馈和知识飞轮

目标：
- 让用户反馈变成产品护城河

能力：
- 每条 prompt 点赞/踩/评论
- 反馈与输入、输出、平台、镜头类型关联
- 高频失败原因统计
- 优秀模板沉淀
- Agent prompt 定期更新

成功标准：
- 能回答哪些类型更容易出片
- 能回答哪些镜头最常失败
- 能基于反馈改进下一轮生成

### Phase 3: 项目化创作

目标：
- 从单次生成升级到短片项目工作台

能力：
- 项目空间
- 角色卡 / 世界观 / 视觉符号库
- 多轮版本管理
- 分镜状态追踪
- 成片素材记录

成功标准：
- 用户可以围绕一个项目持续迭代
- 角色和世界观跨多次生成保持一致

### Phase 4: 商业化和协作

目标：
- 建立可付费的专业创作工具

能力：
- 模板市场
- 团队协作
- 私有知识库
- 高级平台适配
- 导出成生产清单

成功标准：
- 用户愿意为减少试错付费
- 专业模板和项目能力产生留存

## 4. 当前优先级

P0：
- 稳定 V2 导演执行包闭环
- 补齐自动化和 E2E
- 修复前后端契约漂移
- 明确发布门禁

P1：
- 镜头风险标签体系
- 平台适配建议变具体
- 反馈数据结构标准化

P2：
- 模板库
- 项目化工作台
- 知识注入自动化

## 5. 不做什么

当前不做：
- 泛 prompt 市场
- 大而全 AI 工具箱
- 社区画廊
- 排行榜
- 团队协作
- 复杂账号体系
- 商业化套餐

原因：
- 当前最关键的是“出片完成度”，不是功能数量。
