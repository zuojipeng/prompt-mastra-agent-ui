# 镜词 · AI 短片导演执行包最佳实践

目标用户：已经在使用小云雀、Seedance、Kling、Runway、Pika、Sora 等工具，但经常卡在创意不稳、镜头翻车、平台试错成本高的 AI 视频创作者。

## 一句话方法

不要直接把一句创意丢给视频模型。先把创意拆成可执行导演包，再逐镜头验证、标记结果、回收失败原因。

## 推荐工作流

```text
创意输入
  -> 创意体检
  -> 选择重构方向
  -> 生成导演执行包
  -> 逐镜头执行
  -> 标记镜头状态
  -> 反馈失败原因
  -> 用反馈洞察优化下一轮
```

## 1. 输入创意时怎么写

好的输入不需要很长，但要包含三个信息：

- 主体：谁在画面里。
- 场景：发生在哪里。
- 动作或冲突：主体正在做什么，为什么有看点。

示例：

```text
废土小镇里，一个旧清洁机器人守护红裙人偶。
```

不推荐：

```text
帮我做一个很震撼很电影感的科幻短片。
```

原因：主体、动作和可生成画面都太虚，后面很难拆镜头。

## 2. 先看创意体检，不要急着复制 Prompt

创意体检主要回答：

- 这个创意是否适合 AI 视频生成。
- 哪些地方容易翻车。
- 是否需要减少人物、口型、复杂动作或快速运镜。
- 应该往更稳、更风格化还是更电影感的方向改。

如果体检提示风险高，先改创意，不要直接进平台生成。

## 3. 三个重构版本怎么选

- 稳妥版：适合第一次测试，优先降低翻车概率。
- 风格版：适合做账号风格、视觉记忆点、封面感。
- 电影版：适合做完整短片，有起承转合和镜头节奏。

最佳实践：

```text
先选稳妥版跑通，再用风格版或电影版增强表现力。
```

## 4. 分镜卡片怎么用

每张分镜卡都应该当作一次独立生成任务。

执行顺序：

1. 看镜头目的，确认这一镜是建立场景、推进动作还是制造情绪。
2. 看生成模式，决定文生视频、图生视频还是参考图。
3. 看风险标签，提前规避主体漂移、动作复杂、场景跳变。
4. 看稳定性检查，生成前逐项确认。
5. 生成后标记状态：未生成、已生成、翻车、可用。

不要一次把所有分镜合成一个长 Prompt。AI 视频平台更适合逐镜头测试。

## 5. 平台建议怎么用

平台建议不是排名，而是执行策略。

- 中文画面描述、短节奏测试：优先 Seedance / 小云雀。
- 角色一致性强的镜头：先做参考图，再图生视频。
- 复杂运动镜头：先降低运动幅度，再逐步增强。

如果平台结果不稳定，把反馈标成“平台不适配”，不要只靠手工记忆。

## 6. 反馈洞察怎么用

反馈洞察用于回答：

- 哪类镜头最容易翻车。
- 哪个平台差评更多。
- 哪些失败原因最常出现。
- 哪些风险标签需要优先优化。

低样本量时不要下强结论。先积累 5 条以上反馈，再看趋势。

## 推荐 demo 创意

```text
废土小镇里，一个旧清洁机器人守护红裙人偶。
```

这个创意适合演示镜词，因为它同时包含：

- 清晰主体：旧清洁机器人。
- 清晰视觉符号：红裙人偶。
- 清晰场景：废土小镇。
- 适合拆镜头：建立环境、靠近目标、情绪收束。
- 典型风险：主体一致性、复杂动作、场景跳变。
