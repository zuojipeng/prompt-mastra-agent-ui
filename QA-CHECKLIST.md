# QA 验收标准

以下任何一项不通过则阻塞发布。

## 1. 构建检查 (`npm run build`)

- [ ] `next build` 无错误退出
- [ ] 包大小不超过 200 KB (First Load JS)

## 2. 类型检查 (`npx tsc --noEmit`)

- [ ] 源文件（不含 node_modules）无 TypeScript 错误

## 3. 单元测试 (`npm test`)

- [ ] 全部测试通过
- [ ] 覆盖率不低于 80%（核心纯函数）

## 4. 冒烟测试 (`npm run test:smoke`)

- [ ] Stage 1: 静态 HTML 包含关键文本
- [ ] Stage 2: JS bundle 包含所有功能字符串
- [ ] Stage 3: 包大小在合理范围
- [ ] Stage 4: 无 localhost/开发环境泄漏

## 5. E2E API 验收 (`npm run test:e2e`)

- [ ] 核心生成：单镜头、多镜头、带风格 → 通过率 ≥ 90%
- [ ] 高级导演模式：全部 4 个 Bible payload → 通过率 ≥ 90%
- [ ] 镜头优化：2 个 refinement payload → 通过率 100%
- [ ] 全部测试通过率 ≥ 90%

## 6. 手动验收（浏览器）

- [ ] 页面加载不报错（控制台无红色错误）
- [ ] 输入框输入文字，提交后生成提示词
- [ ] 多镜头（3镜/5镜）生成正常
- [ ] 风格选择生效
- [ ] 导演模式展开/收起，填字段后提交
- [ ] 点击「优化此镜头」，输入要求，确认后结果替换
- [ ] 点击「查看原版」，能看到对比，可恢复
- [ ] 复制按钮工作
- [ ] 导出到各平台功能正常
- [ ] 批量导出功能正常
- [ ] 历史记录加载/继续/复制正常
- [ ] 夜间模式正常
