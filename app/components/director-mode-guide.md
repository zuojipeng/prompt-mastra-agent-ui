# 高级导演模式 — 最佳实践示例

## 完整范例：《雨夜追踪》——悬疑短片

### 🎬 一句话创意
```
雨夜街头，一个女孩听见身后脚步声后缓慢回头
```

### 📖 导演模式填写（最佳实践版）

| 字段 | 填写内容 | 为什么这样填 |
|------|---------|------------|
| **主角设定** | 一个穿深灰色连帽雨衣的短发女孩，25岁左右，眼神中带着疲惫和警惕 | 描述要具体到服装颜色、发型、年龄感、情绪状态，AI 才能保持角色一致 |
| **角色任务** | 在雨夜中寻找失踪的弟弟，每次听到脚步声都会警觉回头 | 任务驱动动作，让多镜头之间有叙事逻辑 |
| **世界观** | 近未来中国南方城市，常年下雨，老旧街巷密布霓虹灯牌，地面永远是湿的 | 描述环境的气氛和质感，AI 会根据这些约束生成场景 |
| **视觉符号** | `透明雨伞` `红蓝霓虹倒影在水洼中` | 使用 tag 输入添加，这些元素会在每个镜头中强制出现，形成视觉记忆点 |
| **统一视觉风格** | 赛博朋克 + 王家卫风格混合：饱和的青橙对比色，大量水面的镜面反射 | 风格越具体越好，可以直接参考已有的风格预设 |
| **镜头目的** | 从「建立悬念」到「紧张升级」到「情绪爆发」再到「余韵收尾」 | 为多镜头叙事提供情感弧线 |
| **连续性规则** | `雨量始终保持中雨` `女孩雨衣从不脱掉` `主光源来自霓虹灯` | 这些是 AI 容易犯错的细节，显式约束后一致性更好 |

### 🎯 为什么不建议这样填？

| ❌ 错误示例 | 问题 |
|-----------|------|
| 主角：一个女孩 | 太模糊，AI 无法锁定角色 |
| 任务：找人 | 没有情绪和场景，镜头会缺乏驱动力 |
| 风格：好看 | 主观描述，AI 无法理解 |
| 视觉符号：空 | 浪费了跨镜头一致性的核心能力 |

### 📐 最佳实践原则

1. **主角要「可锁定」** — 有颜色、有服装、有情绪状态
2. **任务要「可演绎」** — 让 AI 知道每个镜头里主角在做什么、为什么做
3. **世界观要「有质感」** — 不仅是「下雨的街道」，而是「什么样的下雨街道」
4. **视觉符号不要超过 3 个** — 太多符号 AI 会顾此失彼
5. **连续性规则锁定易错项** — 不知道写什么就锁「角色外观不变」「光影风格统一」
6. **风格可以混搭** — 如「赛博朋克 + 王家卫」比单一风格更有辨识度
