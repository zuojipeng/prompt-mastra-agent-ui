# Sprint 5 — Prompt 模板管理 + 批量导出 + 差异对比

## 目标
提升用户创作效率：减少重复输入、加快导出流程、看清优化差异

## 计划

### P0: Prompt 模板 + 快速启动
- 输入框下方显示可点击的模板片（如「古风武侠」「赛博都市」「雨中叙事」）
- 点击自动填充输入框和风格/镜头数
- 模板数据在客户端静态定义，无需后端改动

### P0: 批量导出
- 生成结果区增加「批量导出」按钮
- 下拉选择目标平台，一次性复制所有镜头的平台适配文本
- 支持全部平台的批量导出

### P1: Prompt 差异对比
- Refinement 优化后，在 PromptCard 上显示「查看差异」按钮
- 点击展开 inline diff（原版 vs 新版），使用简单的文本对比
- 用户可回退到原版

### P2: 后端改进
- 使用 session 存储时记录 refinement 版本历史
- 部署前端时一并部署后端 fix
